### Алина Лукьянова, разработчик

**Перечень задач с описанием работы**

В рамках проекта создала ORM-модель и полную сущность Category (Категории), реализовала возможность оставлять текстовые отзывы на книги и оценивать их по шкале от 1 до 5 (сущность Review), разработала эндпоинт для отображения страницы книги с детальной информацией (авторы, описание, рейтинги, отзывы). Все реализованные API покрыла автоматическими тестами.

В задачи входило:
- Проектирование и реализация ORM-моделей Category и Review с соответствующими ограничениями и связями
- Создание Pydantic-схем для валидации данных и API-контрактов
- Реализация REST API для работы с категориями (CRUD-операции с правами администратора)
- Реализация REST API для работы с отзывами, позволяющего пользователям оставлять текстовые отзывы и оценивать книги по шкале от 1 до 5 
- Разработка эндпоинта book information для отображения страницы книги с детальной информацией: автор, жанр, категория, описание, средний рейтинг, количество оценок, счетчик текстовых отзывов и вывод 10 свежих отзывов
- Покрытие реализованных сущностей полным набором автоматических тестов: 18 тестов для Category API и 16 тестов для Review API (включая проверку валидации, авторизации, прав доступа, граничных случаев)
- Настройка миграций через Alembic для новых сущностей
- Интеграция API в общую структуру проекта
